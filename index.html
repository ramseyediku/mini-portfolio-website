About

I'm an aspiring developer with a strong focus on front-end technologies and a growing skillset in back-end development. My journey began in junior high with an early interest in coding, which led me to pursue a Computer Science degree and eventually tutor others in web development. I currently build responsive, accessible web applications using SCSS, JavaScript, and React, alongside tools like Redux for state management and React Router for navigation.

I recently completed a full-stack bootcamp where I built and deployed real-world applications using modern technologies like React and Node.js. This experience, combined with hands-on work at UserTesting and a solid foundation in collaborative workflows, has prepared me to contribute meaningfully as a software developer. I'm actively expanding into back-end development with a focus on APIs and server-side logic, aiming to grow into a full-stack role.
